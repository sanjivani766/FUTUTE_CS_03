# FUTUTE_CS_03
API Security Risk Analysis Task 3 – Future Interns

## Overview
**FUTUTE_CS_03** is a repository containing the **API Security Risk Analysis Task 3** conducted as part of the **Future Interns** program. This project focuses on identifying, analyzing, and documenting potential security risks in API endpoints, along with remediation recommendations.

## Contents
- **API Security Risk Analysis REPORT.pdf** – Comprehensive report detailing findings, risk severity, root causes, and recommended remediation steps.
- **screenshots/** – Folder containing relevant screenshots supporting the report and analysis.
- **README.md** – Project overview and repository details.

## Objective
The primary objective of this task is to:
1. Identify potential security vulnerabilities in APIs.
2. Analyze the severity and impact of each finding.
3. Provide actionable recommendations to mitigate risks.
4. Document all findings and evidence systematically for reference.

## Key Features
- Risk identification for common API security issues such as:
  - Unauthenticated Access
  - Missing Object-Level Authorization
  - Excessive Data Exposure
  - Technology Disclosure
- Business-friendly risk summaries
- Detailed root cause analysis and remediation recommendations
- Supporting screenshots for clarity and evidence
